# Weather-Service

This service shows the temperature ,humidity, random weather conditions of specific cities in different geographical location.

## Features

 * ** cities:**  has different cities 

 * ** Temperature:** Shows random temperature between -5 and 35 degrees celcius 

 * ** Humidity: ** shows random humidity between 20 and 90

 * ** Conditions :** shows different weather conditions like sunny ,cloudy which are randomly chosen based on the cities  geographical location 


## Technical Stack
* ** Language : ** Python 3
* ** Modules :random (Standard Library)
* ** Storage :** file system
